# NOTES

## Finding files
* use `:find ..` to find files (with `<tab>` to scroll)
* use `:b ..` to switch between buffers (with `<tab>` to scroll)

## Tags
* `g<C-]>` list matching tags
* `<C-t>`

## Autocomplete
* `<C-n>` `<C-p>` - default
* `<C-x><C-n>`    - just in file
* `<C-x><C-f>`    - filenames
* `<C-x><C-]>`    - tags
* `:h ins-complete`
